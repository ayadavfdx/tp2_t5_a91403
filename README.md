# Gomoku (Five in row)

## Requirements
Python 3.10+

Libraries used:
- **colorama**

## Installation
```bash
pip install colorama 
```

## Description
This project is a console implementation of the classic Gomoku (Five in a Row) game developed in Python as part of an OOP exercise.

## Features
- 10x10 Gomoku board
- Human vs Computer gameplay
- Save game functionality using JSON
- Load saved game from previous session
- Colored console output using **colorama**


## OOP Concepts Applied
- **Encapsulation** 
- **Inheritance**
- **Abstraction**

## How to Run

Run the main file from the terminal:

```bash
python tp2_t5_main.py
